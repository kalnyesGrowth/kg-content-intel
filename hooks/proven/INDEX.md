# Proven Hook Library Index
> Real hooks from real viral videos. Each card includes visual breakdown, psychology, KG adaptation, and Higgsfield prompt.

## How to Use
1. Pick a hook that matches your content angle
2. Read the "Why It Works" section to understand the psychology
3. Adapt using the "KG Adaptation" section
4. Generate the thumbnail/image using the Higgsfield prompt (nano_banana_pro, 1:1, batch 1)
