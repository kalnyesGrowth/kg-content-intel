# Video Script Frameworks
> 8 frameworks with second-by-second breakdowns

## Framework A: THE REVEAL (Highest virality)
**Duration:** 15-25 seconds
**Best for:** Before/after website redesigns, Google ranking improvements

```
[0-2s]  HOOK: "This business was invisible online. Watch what happened."
[2-7s]  BEFORE: Show the old/bad website or Google result
[7-8s]  TRANSITION: Swipe, zoom, or cut
[8-16s] AFTER: Show new website, results, calls, Google ranking
[16-18s] CTA: "DM me 'website' if you want this for your business"
```

Why: Built-in narrative arc. The reveal IS the reason to stay. Highest completion rates.

---

## Framework B: THE BREAKDOWN (Education + Authority)
**Duration:** 20-40 seconds
**Best for:** Educational content, positioning as expert
**Format:** Talking head + screen recording hybrid

```
[0-2s]  HOOK: "Here's why your website isn't getting calls"
[2-7s]  PROBLEM: Identify the specific issue
[7-22s] SOLUTION: Walk through 2-3 fixes with screen recording
[22-27s] PROOF: Show a result or client metric
[27-30s] CTA: "Link in bio" or "Comment AUDIT for a free review"
```

---

## Framework C: THE CLIENT RESULT (Social proof engine)
**Duration:** 15-25 seconds
**Best for:** Building trust, converting warm audience

```
[0-2s]  HOOK: "I just got this text from a client"
[2-5s]  EVIDENCE: Show the text, call log, or review screenshot
[5-15s] CONTEXT: Explain what you did for them
[15-18s] METRIC: "47 calls in 30 days" on screen
[18-20s] CTA: "Want results like this?"
```

Why: Third-party validation > self-promotion.

---

## Framework D: THE PROCESS (Behind-the-scenes)
**Duration:** 20-45 seconds
**Best for:** Showcasing skill, satisfying visual content
**Format:** Screen recording with voiceover or text overlay

```
[0-2s]  HOOK: "Watch me build a website in 60 seconds"
[2-32s] SCREEN RECORDING / TIMELAPSE: Show the actual build
[32-37s] REVEAL: Finished product
[37-40s] CTA: "Comment your business type for a free mockup"
```

---

## Framework E: THE ROAST (Controversy + Education)
**Duration:** 20-35 seconds
**Best for:** Engagement (people LOVE roasts), driving DMs

```
[0-2s]  HOOK: "I found the worst local business website in [city]"
[2-12s] CALLOUT: Show what's wrong (NEVER name the actual business, blur/generic)
[12-22s] FIX: Show how you'd fix each issue
[22-25s] RESULT: "These fixes alone could double their calls"
[25-28s] CTA: "Send me your website and I'll roast it for free"
```

Why: Entertainment + education. Viewers share because "their friend needs to see this."

---

## Framework F: THE STACK (Value ladder / Listicle)
**Duration:** 15-25 seconds
**Best for:** High save rate content

```
[0-2s]  HOOK: "3 things every local business website needs"
[2-7s]  POINT 1: With visual example
[7-12s] POINT 2: With visual example
[12-17s] POINT 3: With visual example
[17-20s] CTA: "Which one is YOUR website missing?"
```

Why: Numbered lists set clear expectations. Easy to rewatch/save.

---

## Framework G: THE STORY (Emotional connection)
**Duration:** 25-45 seconds
**Best for:** Emotional resonance, building parasocial relationship

```
[0-2s]  HOOK: "A restaurant owner called me crying because he was about to close"
[2-12s] SETUP: No online presence, losing to competitors
[12-22s] ACTION: Built website, set up Google, ran local SEO
[22-27s] RESULT: "6 months later, he hired 3 new staff"
[27-30s] LESSON: One-line takeaway
```

Why: Stories > stats. People remember stories 22x better than facts.

---

## Framework H: THE COMPARISON (Us vs. Them)
**Duration:** 15-25 seconds
**Best for:** Justifying price, overcoming "why not just use Wix" objection

```
[0-2s]  HOOK: "What $200 gets you vs. what $800 gets you"
[2-10s] SIDE A: Show the cheap/template option
[10-18s] SIDE B: Show the custom/professional option
[18-21s] KEY DIFFERENCE: One critical gap (mobile, speed, calls)
[21-23s] CTA: "Which one would YOU trust with your money?"
```

Why: Visual comparison is instantly understood. Viewer draws their own conclusion.

---

## Quick Reference: When to Use Each Framework

| Situation | Framework | Duration |
|-----------|-----------|----------|
| Just finished a website | THE REVEAL | 15-25s |
| Teaching something | THE BREAKDOWN | 20-40s |
| Got client results | THE CLIENT RESULT | 15-25s |
| Building live | THE PROCESS | 20-45s |
| Critiquing bad sites | THE ROAST | 20-35s |
| List of tips | THE STACK | 15-25s |
| Client story | THE STORY | 25-45s |
| Price justification | THE COMPARISON | 15-25s |

## Additional Frameworks

### Hook-Value-CTA (Most Versatile)
```
HOOK (1-3s) > VALUE (10-50s) > CTA (3-7s)
```
TikTok: 15-30s total. IG Reels: 30-60s. YouTube Shorts: 50-60s.

### PAS: Problem-Agitate-Solution
```
PROBLEM (3-6s) > AGITATE (5-10s) > SOLUTION (7-15s)
```
Best for: "Your website is costing you customers" style content.

### Hook-Story-Offer (Serge Gatari's Method)
```
HOOK > STORY (brief trust-builder) > OFFER (CTA)
```
Works in any niche. Best for direct-response agency content.

### BAB: Before-After-Bridge
```
BEFORE (5-10s) > AFTER (5-10s) > BRIDGE (10-25s)
```
120% higher engagement vs other formats. Best for redesigns and AI setup results.
