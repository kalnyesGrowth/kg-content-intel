# Thumbnail / Reel Cover Patterns

## Technical Specs
- **Dimension:** 1080 x 1920 (9:16)
- **Safe zone:** Centered 1080 x 1350 px (top/bottom cropped in grid)
- **Text coverage:** Max 30% of visual space
- **Font:** Bold sans-serif ONLY (thin/script fonts invisible on mobile)
- **Contrast:** Must be visible in dark AND light mode

## Pattern 1: The Split Screen
- Left = "before" (ugly website, red X, sad face)
- Right = "after" (beautiful website, green check, happy face)
- Text overlay: "SAME BUSINESS" or the metric ("2 > 47 calls")
- Best for: Before/after reveal content

## Pattern 2: The Bold Text Statement
- Full-bleed background (your face or website screenshot)
- 2-4 word text in bold: "STOP USING WIX" / "WEBSITE ROAST" / "$0 > $10K"
- Best for: Contrarian/controversial content

## Pattern 3: The Number Lead
- Giant number dominates the frame: "97%" / "47 CALLS" / "$400"
- Small supporting text below
- Best for: Stat-driven or result-driven content

## Pattern 4: The Reaction Face
- You looking shocked/excited at a phone or laptop screen
- Arrow or circle pointing to what you're reacting to
- Best for: Client result reveals, "I just..." hooks

## Pattern 5: The Screen Grab
- Screenshot of an actual website (good or bad) with annotation arrows
- Text overlay: "Fix this NOW" or "This is why you're losing customers"
- Best for: Roast/breakdown/educational content

## Design Rules
- Custom covers increase CTR by up to 40% vs auto-generated
- Use "Emotional Timestamp Targeting": set cover to the frame with strongest emotional impact
- Consistent template set across all covers for cohesive grid
- Use burned-in captions in video: 25% higher completion rate
- 85% of views happen without sound. Captions are mandatory.
